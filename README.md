# YouTube Score Video Dataset

This is the official dataset repository for `YouTube Score Video Dataset` from following paper
> U-MusT: A Unified Framework for Cross-modal Translation of Score Images, Symbolic Music, and Performance Audio
<!-- > [https://dl.acm.org/doi/10.1145/3715159](https://dl.acm.org/doi/10.1145/3715159) -->

This repository contains:
* Metadata of score following videos from YouTube in csv format
* Scripts related to slide segmentation
* Scripts related to system croping and resizing
* Scripts related to audio/image tokenizing
* Checkpoints of following models
    * YOLO-system (YOLOv8m finetuned on musical system detection)
    * YOLO-staff (YOLOv8m finetuned on staff height detection)
    * RQVAE model for image tokenization
    * DAC model for audio tokenization
